옷장요정의 **유스케이스 다이어그램**을 모아둔 문서 사이트입니다.
다이어그램은 [PlantUML](https://plantuml.com/ko/use-case-diagram)로 작성하며, Astro Starlight로 렌더링합니다.

## 시작하기

- 왼쪽 사이드바의 **유스케이스** 문서들을 확인하세요.
- 새 다이어그램은 저장소 루트의 `TEMPLATE.md`를 복사해 `src/content/docs/use-cases/`에 추가합니다.

## 로컬 실행

```bash
npm install
npm run dev   # http://localhost:4321
```
